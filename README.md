# RSS Reader

https://rss-reader-x.vercel.app/

## Commands

```bash
# Build production site.
npm run build

# Check for linting errors.
npm run check

# Deploy to Preview environment.
npm run deploy

# Depoy to Production environment.
npm run deploy -- --prod

# Format source files.
npm run format

# Start the development server and watch for changes.
npm start
```
